# Guess Number Game

This is a simple number guessing game written in Python.  
It is a practice project from Angela Yu's Python Bootcamp (Day 12).

## Description
- The program generates a random number between 1 and 100.
- The player tries to guess the number.
- The program will tell the player:
  - "Too high"
  - "Too low"
- The game ends when the correct number is guessed.

## Difficulty Levels
- Easy: 10 attempts
- Hard: 5 attempts

## How to Run
Make sure Python is installed, then run:

```bash
python guess_number_game.py
